# Todo アプリケーション

## 概要
ai-template-labのテンプレートを使用して開発したシンプルなTodoアプリケーション

## 技術スタック
- フロントエンド: HTML5, CSS3, Vanilla JavaScript
- データ保存: LocalStorage
- テンプレート: ai-template-lab v1.0

## 機能
- ✅ タスクの追加
- ✅ タスクの完了/未完了切り替え
- ✅ タスクの削除
- ✅ ローカルストレージへの自動保存
- ✅ 操作ログの記録

## セットアップ
```bash
# プロジェクトディレクトリに移動
cd custom/projects/todo-app/src/

# ローカルサーバーを起動
python3 -m http.server 8080

# ブラウザで開く
# http://localhost:8080
```
